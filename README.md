### Приложение для просмотра книг

![image](https://github.com/8Alex/book-app/assets/91260590/19cd2c58-3353-4576-8676-d2f39eb81083)

#### Используемые технологии:

- React
- Redux
- TypeScript
- JavaScript
- SCSS
- HTML5/CSS3

#### Запуск проекта:

```
git clone https://github.com/8Alex/book-app.git
cd book-app
npm install
npm run dev
```
