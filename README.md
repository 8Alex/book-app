### Приложение для просмотра книг

![image](https://github.com/8Alex/book-app/assets/91260590/d4e87fcf-0aca-4c0f-9d85-aa9beca600e3)

#### Используемые технологии:

- React
- Redux
- Typescript
- JavaScript
- SCSS
- HTML5/CSS3

#### Запуск проекта:

```
git clone https://github.com/8Alex/book-app.git
cd book-app
npm install
npm run dev
```
